# MaterialBinLoader

MaterialBinLoader enables you to use RenderDragon Shaders in Minecraft.

* libMaterialBinLoader mod installed

## Installation

1. Download libMaterialBinLoader from [Releases](https://github.com/OpenMCBE/LeviLauncher-MaterialBinLoader/releases) or [GitHub Actions](https://github.com/OpenMCBE/LeviLauncher-MaterialBinLoader/actions)
2. Install LeviLauncher
3. Add libMaterialBinLoader mod in LeviLauncher
4. Launch Minecraft through LeviLauncher
